# Doctor Appointment API  
**Web Services Course Project**

## Overview
The Doctor Appointment API is a RESTful web service developed as part of a Web Services course.  
The project demonstrates API design, implementation, documentation, and testing concepts using a real-world clinic scheduling use case.

The service supports managing doctors and appointments, including booking, updating, and canceling appointments.

---

## Project Objectives
- Design and implement RESTful web services
- Define and document an API contract using Swagger (OpenAPI)
- Perform functional and negative API testing
- Automate selected API tests
- Demonstrate teamwork and source control using GitHub
